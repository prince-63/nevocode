## The switch Statement â€“ Handling Multiple Cases
If you have multiple choices (like a restaurant menu), `switch` makes it cleaner.

<CodeBlock language="c">
{
`#include <iostream>
using namespace std;

int main() {
    int day = 3;
    switch (day) {
        case 1: cout << "Monday"; break;
        case 2: cout << "Tuesday"; break;
        case 3: cout << "Wednesday"; break;
        default: cout << "Invalid day!";
    }
    return 0;
}`
}
</CodeBlock>

ğŸ“Œ **Output:**  
```
Wednesday
```

## The while Loop â€“ Repeat Until a Condition is Met
Think of a car moving until it runs out of fuel.

<CodeBlock language="c">
{
`#include <iostream>
using namespace std;

int main() {
    int fuel = 5;
    while (fuel > 0) {
        cout << "Driving... Fuel left: " << fuel << endl;
        fuel--;
    }
    cout << "Out of fuel!";
    return 0;
}`
}
</CodeBlock>

ğŸ“Œ **Output:**  
```
Driving... Fuel left: 5
 Driving... Fuel left: 4
 Driving... Fuel left: 3
 Driving... Fuel left: 2
 Driving... Fuel left: 1
 Out of fuel!
```
