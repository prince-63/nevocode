A **variable** in C++ is a name given to a memory location where data is stored. Every variable has a **data type**, which determines the kind of value it can store.

## **Declaring and Initializing Variables**

Variables in C++ must be declared before use. The general syntax is:

<CodeBlock language="c">{`data_type variable_name = value;`}</CodeBlock>

**Example:**

<CodeBlock language="c">
{
`#include <iostream>
using namespace std;

int main() {
int age = 25;
float pi = 3.14;
char grade = 'A';
string name = "prince";

    cout << "Age: " << age << endl;
    cout << "PI: " << pi << endl;
    cout << "Grade: " << grade << endl;
    cout << "Name: " << name << endl;

    return 0;

}`
}

</CodeBlock>

ğŸ“Œ **Output:**

```c
Age: 25
 PI: 3.14
 Grade: A
 Name: prince
```

### **Multiple Variable Declarations**

  {`int a = 10, b = 20, c = 30;
float x = 5.5, y = 10.5;`}
</CodeBlock>

## Data Types in C++ ğŸ“Š

C++ provides several built-in data types:

| Data Type   | Description                                    | Size (32-bit) | Size (64-bit) |
| ----------- | ---------------------------------------------- | ------------- | ------------- |
| `int`       | Stores whole numbers                           | 4 bytes       | 8 bytes       |
| `float`     | Stores decimal numbers (single-precision)      | 4 bytes       | 4 bytes       |
| `double`    | Stores double-precision floating-point numbers | 8 bytes       | 8 bytes       |
| `char`      | Stores a single character                      | 1 byte        | 1 byte        |
| `string`    | Stores multiple characters (dynamic size)      | Varies        | Varies        |
| `void`      | Represents an absence of data                  | N/A           | N/A           |
| `long`      | Extended size for integer values               | 8 bytes       | 8 bytes       |
| `long long` | Stores larger integer values                   | 16 bytes      | 16 bytes      |

### **Size of Different Data Types**

The `sizeof` operator can be used to determine the memory size of any data type:

<CodeBlock language="c">
{
`#include <iostream>
using namespace std;

int main() {

    cout << "Size of int: " <<  sizeof(int) << endl;
    cout << "Size of float: " <<  sizeof(float) << endl;
    cout << "Size of char: " <<  sizeof(char) << endl;
    cout << "Size of long long int: " <<  sizeof(long long int) << endl;
    cout << "Size of double: " <<  sizeof(double) << endl;
    cout << "Size of string: " <<  sizeof(string) << endl;

    return 0;

}`
}

</CodeBlock>

ğŸ“Œ **Output:**

```c
Size of int: 4
 Size of float: 4
 Size of char: 1
 Size of long long int: 8
 Size of double: 8
 Size of string: 32
```

## Constants and Keywords ğŸ”’

Example of **constant declaration**:

<CodeBlock language="c">
  {`#define PI 3.1415

### **Using Constants in a Program**

<CodeBlock language="c">
{
`#include <iostream>

    const int radius = 5;

    return 0;

}`
}

</CodeBlock>
