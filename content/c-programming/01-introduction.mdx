## What is C? 
C is a **general-purpose, procedural programming language** developed by **Dennis Ritchie** in **1972** at Bell Labs. It remains one of the most influential programming languages, known for its efficiency, portability, and direct access to system hardware. C is widely used for **system programming, operating systems, embedded systems, and high-performance applications**.

## Why Learn C? üßê
- **Foundation for Other Languages** ‚Äì C has influenced many modern languages like **C++, Java, Python, and Go**.
- **High Performance** ‚Äì C is used in systems where speed is crucial, such as **operating system kernels, databases, and embedded systems**.
- **Low-Level Access** ‚Äì It provides direct memory manipulation using **pointers**, making it ideal for hardware-level programming.
- **Portability** ‚Äì C programs can be compiled and executed on different architectures with minimal modifications.
- **Real-World Applications** ‚Äì C is used in **Linux kernel, MySQL database, Git, embedded systems, and game engines**.

## Writing Your First C Program ‚úçÔ∏è
Let's write a simple **Hello, World!** program in C.

<CodeBlock language="c">
{
`#include <stdio.h>

int main() {
    printf("Hello, World!"); // output: Hello World!
    return 0;
}`
}
</CodeBlock>

### **Explanation:**
1. `#include <stdio.h>` ‚Äì Includes the standard input/output library.
2. `main()` ‚Äì The entry point of a C program.
3. `printf()` ‚Äì Prints output to the console.
4. `return 0;` ‚Äì Indicates successful execution.

### **Compiling & Running the Program**
#### **For Windows:**
<CodeBlock language="bash">
{`gcc hello.c -o hello.exe
hello.exe`}
</CodeBlock>

#### **For macOS/Linux:**
<CodeBlock language="bash">
{`gcc hello.c -o hello
./hello`}
</CodeBlock>

## Understanding C Compilation Process ‚öôÔ∏è
C programs undergo the following steps:

1. **Preprocessing** ‚Äì Handles macros, `#include` directives, and other preprocessor commands.
2. **Compilation** ‚Äì Converts C code into assembly language.
3. **Assembly** ‚Äì Translates assembly code into machine code (binary format).
4. **Linking** ‚Äì Combines multiple object files into an executable program.

### **Why Compilation Matters?**
Unlike interpreted languages like Python, **C requires compilation** to generate machine-executable code, making it much faster in execution. Understanding this process will help you debug and optimize your programs effectively.
