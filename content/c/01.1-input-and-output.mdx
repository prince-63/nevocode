## **Introduction**

Input and output (I/O) operations allow programs to interact with users. In **C programming**, input refers to taking data from the user, while output refers to displaying information on the screen. The **stdio.h** library provides essential functions like `printf()` for output and `scanf()` for input, making it easy to communicate with users.

## **Taking Input in C**

To take user input, we use the `scanf()` function. It reads data from the keyboard and stores it in a variable. The syntax is simple:

<CodeBlock language="c">
  {`scanf("format specifier", &variable);
`}
</CodeBlock>

For example, if we want to take an integer input from the user and store it in a variable, we write:

<CodeBlock language="c">
{`#include <stdio.h>

int main() {
int age;
printf("Enter your age: ");
scanf("%d", &age);
printf("You entered: %d\\n", age);
return 0;
}
`}

</CodeBlock>

Here, `scanf("%d", &age);` reads an integer and stores it in `age`. The `&` (address-of operator) is required to assign the input value correctly.

## **Displaying Output in C**

For output, C provides the `printf()` function. It allows us to print text and variables on the screen. The syntax is:

<CodeBlock language="c">
  {`printf("format specifier", variable);
`}
</CodeBlock>

Let's print a number using `printf()`:

<CodeBlock language="c">
{`#include <stdio.h>

int main() {
int num = 10;
printf("The number is: %d\\n", num);
return 0;
}
`}

</CodeBlock>

```
The number is: 10
```

The `%d` format specifier tells `printf()` to print an integer value.

## **Common Format Specifiers in C**

C uses format specifiers to define the type of data being printed or read. Here are some commonly used ones:

- `%d` or `%i` for integers
- `%f` for floating-point numbers
- `%c` for characters
- `%s` for strings
- `%lf` for double precision floating-point numbers

Using the right format specifier ensures correct input and output handling.

## **Taking Multiple Inputs**

`scanf()` allows us to take multiple inputs at once. We separate format specifiers using spaces.

<CodeBlock language="c">
{`#include <stdio.h>

int main() {
int a, b;
printf("Enter two numbers: ");
scanf("%d %d", &a, &b);
printf("You entered: %d and %d\\n", a, b);
return 0;
}
`}

</CodeBlock>

If the user enters **5 10**, the program stores **5** in `a` and **10** in `b`, then prints them.

## **Handling Character Input and Output**

For single-character input, we use `getchar()`, and for output, we use `putchar()`.

<CodeBlock language="c">
{`#include <stdio.h>

int main() {
char ch;
printf("Enter a character: ");
ch = getchar();
printf("You entered: ");
putchar(ch);
return 0;
}
`}

</CodeBlock>

This program takes a character as input and prints it on the screen.

## **Why Input & Output Matters?**

Understanding input and output operations in C is fundamental for writing interactive programs. Functions like `printf()`, `scanf()`, `getchar()`, and `putchar()` make it easy to communicate with users. By experimenting with different input and output formats, you can enhance your programming skills and create dynamic applications.
